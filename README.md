Agenda

## Algoritmo do Exercício

    INÍCIO
        VAR
        dado : conjunto de cadeia
        i, j, k: inteiros
        x : cadeia

        PARA i de 1 até 10 passo 1, FAÇA
            ESCREVA('Nome: ")
            dado[i,1]
            ESCREVA('Endereço: ")
            dado[i,2]
            ESCREVA('Código Postal: ")
            dado[i,3]
            ESCREVA('Bairro: ")
            dado[i,4]
            ESCREVA('Telefone: ")
            dado[i,5]
        FIM PARA
        PARA i de 1 até 10 passo 1, FAÇA
            PARA j de 1 até 5 passo 1, FAÇA
                ESCREVA dado [i,j]
            FIM PARA
        FIM PARA
    FIM
